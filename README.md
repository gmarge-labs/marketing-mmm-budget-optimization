# marketing-mmm-budget-optimization
Bayesian Media Mix Modeling (MMM) using PyMC for marketing budget optimization

This project implements a Bayesian Media Mix Model using PyMC to measure the true impact of marketing channels and optimize budget allocation.

Key Features
Adstock transformation to capture carryover effects
Saturation modeling to capture diminishing returns
Channel contribution estimation
Budget optimization using marginal returns
Tools

Python, PyMC, Pandas, NumPy
